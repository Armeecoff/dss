# Telegram Archivarius Bot

Telegram-бот «Архивариус» — перехватчик удалённых/изменённых сообщений, одноразовых медиа, голосовых и кружочков. Работает в режиме Telegram Business.

## Запуск

```
python main.py
```

## Переменные окружения

| Переменная | Описание | Обязательная |
|---|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather | ✅ |
| `ADMIN_ID` | Telegram ID первого администратора | ✅ |
| `YOOKASSA_SHOP_ID` | ID магазина ЮКассы | Для оплаты рублями |
| `YOOKASSA_SECRET_KEY` | Секретный ключ ЮКассы | Для оплаты рублями |
| `BOT_USERNAME` | Username бота (без @) | Рекомендуется |

## Структура проекта

- `main.py` — точка входа
- `bot.py` — хендлеры бота, логика бизнес-режима
- `admin.py` — панель администратора
- `payments.py` — интеграция с ЮКассой и Telegram Stars
- `database.py` — работа с SQLite базой данных
- `bot_data.db` — база данных (создаётся автоматически)

## Возможности бота

- Перехват удалённых сообщений (через Business API)
- Перехват изменённых сообщений
- Сохранение одноразовых фото/видео
- Сохранение голосовых и кружочков
- Подписка: включается/выключается из админки
- Оплата через ЮКассу (рубли) и Telegram Stars
- Гибкие тарифные планы (любые сроки и цены)
- Рассылка пользователям
